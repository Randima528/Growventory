# 🪴 Growventory – Simple Inventory Manager (Plants)

**Growventory** is a lightweight and user-friendly web-based inventory management system designed for small businesses to efficiently manage plant stock.  
The system allows users to **add, view, update, and delete products**, with **secure login and registration functionality** for user authentication.

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Backend:** Pure Node.js (without Express)  
- **Database:** MySQL  

---

## 🌿 Key Features
- 🔐 User authentication with login and registration pages  
- ✅ CRUD operations for managing plant inventory  
- 💻 Simple, intuitive, and responsive user interface  
- 🔗 RESTful API integration between frontend and backend  
- ⚡ Lightweight architecture for easy deployment and scalability  

---

## 🚀 Project Overview
Built as an **individual project** to strengthen **full-stack development** skills using **pure Node.js** and **React.js**.  
The system demonstrates a complete workflow from frontend design to backend logic and database management, emphasizing secure and efficient business operations.
